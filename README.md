# GT-QuantumTransition
格雷：量子跃迁(GregTech:QuantumTransition)是一款围绕GTCEu,TJcore,GregicalityMultiblocks展开，配合NuclearCraft: Overhauled,Quantum Minecraft Dynamics,Industrial Craft 2等其他主流模组联动魔改的大型魔改整合包。整合包围绕科技，魔法，冒险等维度制作了约1600+（还在更新）拥有完整流程，介绍详细的任务，将完全覆盖你的整个游戏阶段。你将在游戏里研究16级电压的主线科技，8级支线科技，4条并联魔法任务组,总计30章任务。探索超过30个星球维度，以及100+随机遗迹，200+生物群系，以及遍布世界的冒险地牢，新村庄，藏宝地，探索全新机制生产的洞穴。
